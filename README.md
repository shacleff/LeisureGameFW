# LeisureGameFW

## 项目介绍
本项目是个人在游戏开发中总结出来的在Unity3D项目开发中常用到的功能和模块，集合了各种可敏捷开发，重用开发的功能。

## 项目功能 

### 功能模块及UI基础框架
    1. 主界面
    2. 游戏中界面
    3. 设置界面
    4. 商店界面
    5. 礼包（倒计时）
    6. 离线功能（离线界面以及倒计时）
    7. 时间管理
    8. 广告sdk功能

### 2.游戏管理类（基于单例）
	1.GameManager
	2.UIManager
	3.EntityManager
	4.ShopManager
	5.ScoreManager
	6.SoundManager
